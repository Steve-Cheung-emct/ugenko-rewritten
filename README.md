# ugenko-rewritten
基於 TikZ 水印的 upLaTeX genko (原稿紙) 的重寫

本項目使用 八登 崇之 的 jfmutil.pl 工具 複製的虛擬字体。
刪除 glue 表 和 kern 表，用於 一字一格的作文紙。
